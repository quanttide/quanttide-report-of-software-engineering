# AGENTS — 报告写作规范

## 报告类型

本仓库记录两类内容，由文件结构和写作方式区分：

### 决策笔记 `platform/`

记录**方向判断和决策推演**，不是实验报告。

写作规则：
- 直接进入命题，不写背景铺垫（读者已有项目上下文）
- 数据只作为论据引用，不展开实验细节
- 不用 mermaid 流程图、编号章节、规范化表格
- 核心是"为什么这重要"，不是"发现了什么"
- 篇幅分配：理由 ≈ 70%，数据 ≈ 30%
- 格式保持探索笔记风格，不包装为正式发布件

### 实验报告 `experiments/`（待定）

当需要记录实验方法、数据、可复现步骤时使用。
